# Triton 示例

**刚开始接触 Triton 推理服务器？** 使用[这些教程](https://github.com/triton-inference-server/tutorials)开启您的 Triton 之旅！

本文件夹包含以下内容：
* jetson：涵盖在 Jetson 设备上部署 Triton 推理服务器的内容。
* model_repository：这是一个用于使用 Triton 推理服务器部署模型的基础模型仓库。
